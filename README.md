# Extra Chill Admin Tools

This plugin is retired. It intentionally boots as an inert shell for one release cycle so the deployed package can be upgraded before network deactivation and repository archival.

The former network-admin application, ability registrations, handlers, and frontend dependencies have been removed. Domain plugins now own the capabilities; this repository must not be used as a compatibility layer.

See [the retirement runbook](docs/retirement.md) for prerequisites, merge order, removed contracts, retained compatibility, and disablement verification.

Historical release notes remain in [docs/CHANGELOG.md](docs/CHANGELOG.md).
